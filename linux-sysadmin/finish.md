# 這十章在實際工作裡怎麼配合
<br>

## 每一章負責什麼
<br>

| # | 章節 | 它回答的問題 | 核心指令 |
| :--- | :--- | :--- | :--- |
| 1 | 定位與觀察 | 我在哪個目錄？這裡有哪些檔案？ | `pwd` `ls -l` `cd` |
| 2 | 搜尋 | 我要找的檔案在哪個目錄？ | `find` `grep` |
| 3 | 權限 | 誰可以讀寫這個檔案？ | `ls -l` `chmod` `chown` |
| 4 | 帳號 | 誰可以登入這台機器？ | `/etc/passwd` `usermod` |
| 5 | 服務 | 該執行的服務有沒有在執行？ | `systemctl` |
| 6 | 日誌 | 剛才發生了什麼？ | `journalctl` `grep` |
| 7 | 套件 | 裝的是哪一個版本？ | `apt policy` |
| 8 | 網路 | 對外開放了哪些連接埠？ | `ss` `ufw` |
| 9 | 磁碟 | 磁碟還剩多少空間？ | `df` `du` |
| 10 | 綜合 | 從哪裡查起？ | 以上全部 |

---
<br>

## 實際排查時的順序
<br>

```text
           有人說「系統打不開」
                    │
                    ▼
        ┌───────────────────────┐
        │  第 5 章：服務在跑嗎？  │
        └───────────┬───────────┘
                    │
        ┌───────────┴───────────┐
        ▼                       ▼
     在跑                     沒在跑
        │                       │
        │              ┌────────▼────────┐
        │              │ 第 6 章：紀錄裡  │
        │              │ 寫了什麼原因？   │
        │              └────────┬────────┘
        │                       │
        │        ┌──────────────┼──────────────┐
        │        ▼              ▼              ▼
        │   Permission     No space left   Address in use
        │     denied                              │
        │        │              │                 │
        │        ▼              ▼                 ▼
        │   第 3 章權限     第 9 章磁碟      第 8 章連接埠
        │
        ▼
   服務正常，但使用者仍打不開
        │
        ├──▶ 第 3 章：那個使用者有沒有權限？
        └──▶ 第 4 章：他的帳號有沒有問題？
```
